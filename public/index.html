共有AR空間 - 最小モデル

複数の端末で同じHiroマーカーをカメラに映すと、 マーカー位置を共通原点として各端末の現実位置が仮想空間内で同期されます。 入室時には各アバターがポップアップ表示されます。

使い方

  1. 下のHiroマーカーを別の画面に表示するか印刷します。
  2. 2台以上のスマートフォンで下のボタンから入室します。
  3. 各端末でマーカーをカメラに映してください。
  4. 他の端末の位置に色付きアバターが表示され、移動に追随します。
注意: カメラ利用には HTTPS 接続が必要です。 ローカルテストでは ngrok http 3000 等でトンネルを作成してください。
